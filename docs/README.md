# docs/

**Status:** reserved. Phase 4 — **MkDocs-Material site not yet stood up.**

## What will land here

- `mkdocs.yml`           — site config (Material theme, mermaid, search, search suggestions).
- `index.md`             — landing: what Trinary + TriPy are, 30-second quick start.
- `engine/`              — C ABI reference (autogenerated from `engine/include/trinary/trinary.h`).
- `language/`            — `.tri` language reference (grammar, stdlib, examples).
- `tripy/`               — Python front-end reference (APIs, `@accelerate`, CLI).
- `benchmarks/`          — performance claims with reproducible recipes.
- `adr/`                 — one Architecture Decision Record per D-row in the architecture MD.
- `security.md`          — threat model and disclosure policy.

## Contract

1. Every public symbol in `trinary.h` and `tripy/_core.pyi` has a doc page.
2. Every performance claim is backed by a `benchmarks/` recipe reproducible
   in one command (see `benchmarks/README.md`).
3. Docs build is a CI job; broken links fail the PR.

## Why this directory exists now if it's empty

§2 of the architecture doc mandates this path. Doc work before the engine is
complete would churn; doc work after the engine stabilizes will be focused
and permanent.
